• Android Debug Bridge

         Google LLC

    Android Debug Bridge 是一个开源的 Android 开发工具，它是 Android Open Source Project (AOSP) 中的一部分。AOSP 采用了 Apache 2.0 许可协议。

  • Scrcpy

         Genymobile

    Scrcpy 是一款用于在 PC/Linux/Mac 中查看 Android 设备屏幕并控制 Android 设备的开源软件，采用 Apache 2.0 许可协议。

  • Sun-Valley-ttk-theme

         Sun-Valley-ttk-theme

    Sun-Valley-ttk-theme 是一款开源的 Tkinter/ttk 主题，基于 Sun Valley 视觉风格。采用 MIT 许可协议。
